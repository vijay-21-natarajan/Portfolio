Social Media Web App

This project is a scalable social media web platform modeled after common features found in popular social networks. It supports essential functionalities such as user registration and login, user profile management, and an interactive homepage feed.

One of the core technical highlights is the friend suggestion system, which uses graph data structures and adjacency lists to identify mutual connections and recommend new friends to users.

The backend is structured using Flask and modular Python scripts to manage routes, models, and utilities. Data models define user relationships, account metadata, and message storage.

The frontend is likely served with HTML/CSS templates and uses AJAX or similar technologies for dynamic updates. The platform aims to simulate real-world social behavior online, offering a clean and functional interface, reliable session management, and real-time features like messaging.
